See the main README in the "Selenium Automation" directory for instructions on how to run these examples. 

This program will pull grades from a tab-separated file and enters them into the corresponding link in the grade book by graded event title (e.g. "Bonus Points", "WPR1") in AMS. Here's an example of grades.tsv:

Quiz1	Quiz2
10	15
12	13
15	15
...

The program will open AMS and then ask you to navigate to the Grade Book for the course. Once you're there, the program will continue.
