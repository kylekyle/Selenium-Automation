Given a roster of cadet names, this program will pull the following fields into a TSV file:

 * Name 
 * Sex 
 * Primary Duty 
 * Major 
 * 2nd Maj
 * CQPA
 * Basic Branch 
 * USMA ID 
 * Class 
 * Phone
 * Eligible for Travel?
 
Here is an example of the format for the roster:

King	Kyle
Sagisi	Joeseph
