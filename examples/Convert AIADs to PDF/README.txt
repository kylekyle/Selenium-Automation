See the main README in the "Selenium Automation" directory for instructions on how to run these examples. 

This program will generate a PDF of all the current AIADs in AMS. 

Additional dependencies: 

* wkhtmltox-0.12.4_msvc2015-win64.exe
* The AIAD app in AMS

The PDF will be saved to the PDFs directory. You can select them all, right click, and click "Combine using Acrobat".
